## 📖 项目介绍

**赛博农村大集** 是一个将传统农村大集与现代互联网技术相结合的农产品电商平台。我们致力于为农村生产者提供一个便捷的商品销售渠道，同时让城市消费者能够便捷购买到新鲜、地道的农产品。

平台的核心理念是「土酷」（Tu-Cool），将传统农村大集的粗犷、鲜活、人情味与赛博朋克的高科技、霓虹风格相融合，打造独特的视觉体验。在这里，你可以感受到乡土气息与科技感的奇妙碰撞。

### 🌟 核心口号

> 连接未来说，只卖土特产

## 🚀 功能特点

### 🧑‍🌾 商家端功能

- **商品发布**：支持上传商品图片、填写商品名称、描述、价格等信息
- **分类管理**：商品自动归类到对应分区（东头-蔬菜、西头-肉蛋、老树下-干货等）
- **库存管理**：实时更新商品库存状态
- **订单管理**：查看订单列表，更新订单状态

### 🛒 客户端功能

- **商品浏览**：瀑布流展示所有在售商品，支持分类筛选
- **购物车**：将心仪商品「装进蛇皮袋」，随时查看和管理
- **购买流程**：简洁的购买流程，模拟支付功能
- **订单记录**：查看历史订单，了解购买状态

### 🎨 特色功能

- **赛博水印**：商品图片自动添加带有科技感的水印
- **方言描述**：支持语音描述，保留乡土气息
- **霓虹UI**：独特的赛博朋克视觉风格
- **本地存储**：数据保存在浏览器本地，无需后端即可使用

## 🛠️ 技术栈

- **前端框架**：Vue 3（CDN引入）
- **样式**：原生CSS + CSS变量
- **数据存储**：LocalStorage（浏览器本地存储）
- **部署方式**：静态页面，可直接部署到任意静态托管服务

### 兼容浏览器

- Chrome / Edge / Firefox / Safari 最新版本
- 移动端浏览器（iOS Safari、Android Chrome等）

## 📱 使用指南

### 快速开始

1. **克隆项目**
   ```bash
   git clone https://github.com/你的用户名/cyber-rural-fair.git
   ```

2. **打开项目**
   - 直接用浏览器打开 `index.html` 文件
   - 或使用本地服务器：`npx serve` 或 `python -m http.server`

3. **开始使用**
   - 点击「支棱起来」成为商家发布商品
   - 在首页浏览并购买心仪农产品

### 买家使用流程

1. 打开平台首页，浏览或在分类中寻找心仪商品
2. 点击商品查看详情
3. 点击「装进蛇皮袋」加入购物车
4. 进入购物车确认商品和数量
5. 点击「结算」创建订单
6. 等待收货，享受新鲜农产品

### 卖家使用流程

1. 点击底部「支棱起来」进入发布页面
2. 拍摄或上传商品照片
3. 填写商品名称、描述和价格
4. 选择商品分类
5. 点击发布，商品即上架销售
6. 在订单页面查看和管理销售订单
