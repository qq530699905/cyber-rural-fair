# 赛博农村大集用户手册

## 一、入门指南

### 1.1 什么是赛博农村大集？

赛博农村大集是一个将传统农村大集搬到线上的农产品交易平台。在这里，农户可以直接销售自己种植、养殖的农产品，城市居民可以便捷购买到新鲜、地道的土特产。平台采用独特的“土酷”设计风格，将赛博朋克元素与乡土气息相融合，为用户带来全新的购物体验。

### 1.2 如何访问平台？

您可以通过以下方式访问赛博农村大集：

1. **直接打开**：在浏览器中打开项目目录下的 `index.html` 文件即可使用全部功能
2. **本地服务器**：在项目目录运行 `npx serve` 或 `python -m http.server`，然后在浏览器访问显示的地址
3. **在线部署**：将项目部署到任意静态网页托管服务（如GitHub Pages、Vercel等），通过互联网访问

### 1.3 需要准备什么？

使用赛博农村大集，您只需要准备以下物品：

- **设备**：电脑、平板或智能手机（推荐使用移动端获得最佳体验）
- **浏览器**：Chrome、Edge、Firefox、Safari等现代浏览器
- **网络**：稳定的互联网连接（用于加载页面图片等资源）
- **图片**：如果您是卖家，需要准备商品图片（支持JPG、PNG等常见格式）

## 二、买家使用教程

### 2.1 浏览商品

**第一步：进入首页**

打开平台后，您会看到首页展示的所有在售商品。页面采用瀑布流布局，商品卡片依次排列，您可以向下滚动浏览更多商品。

**第二步：选择分类**

如果您有明确的购物目标，可以点击顶部的分类按钮快速筛选。平台提供以下分类：

- 全部好货：展示所有商品
- 🥬 东头-蔬菜：各类新鲜蔬菜
- 🍎 水果摊：时令水果
- 🥚 西头-肉蛋：肉类、禽蛋等
- 🍄 老树下-干货：干货、腌制品等
- 🎨 手艺摊：手工制作的物品

**第三步：查看详情**

点击任意商品卡片，可以查看商品的详细信息，包括大图、详细描述、价格和库存等。

### 2.2 购买商品

**方式一：加入购物车**

1. 在商品列表页面，点击「装进蛇皮袋」按钮
2. 商品会被添加到购物车中
3. 页面会显示提示信息“装进蛇皮袋啦！”
4. 您可以继续浏览其他商品，或点击底部导航栏中间的「蛇皮袋」图标查看购物车

**方式二：立即购买**

1. 在商品列表页面，点击「立即带走」按钮
2. 商品会被添加到购物车
3. 购物车侧边栏会自动打开，方便您直接结算

### 2.3 结算订单

1. 在购物车页面，确认已选商品和数量
2. 如需修改数量，点击数量旁边的「+」或「-」按钮
3. 如需删除商品，点击商品右下角的垃圾桶图标
4. 确认商品无误后，点击「💰 结算」按钮
5. 订单创建成功！系统会显示提示信息

### 2.4 查看订单

1. 点击底部导航栏最右边的「订单」图标
2. 在订单页面，您可以查看所有历史订单
3. 每个订单显示：订单号、商品列表、总金额、订单状态
4. 订单状态说明：
   - 「待处理」：订单已创建，等待确认收货
   - 「已完成」：已确认收货，交易完成

### 2.5 确认收货

收到购买的农产品后，请在订单页面点击「确认收货」按钮，完成交易闭环。

## 三、卖家使用教程

### 3.1 成为卖家

1. 点击页面底部的「支棱起来」按钮，或点击顶部导航栏的「支棱起来」按钮
2. 您将进入商品发布页面
3. 填写完商品信息并发布后，您就正式成为卖家啦！

### 3.2 发布商品

**第一步：上传商品图片**

1. 点击「商品图片」区域的虚线框
2. 在弹出的文件选择器中，选择您准备好的商品图片
3. 图片会自动预览显示
4. 建议使用清晰的实物拍摄图，避免使用网图

**第二步：填写商品名称**

在「商品名称」输入框中填写您的商品名称。例如：“正宗土鸡蛋”、“新鲜蔬菜大礼包”等。名称应该简洁明了，让买家一眼就能看出是什么商品。

**第三步：选择分类**

在「商品分类」下拉菜单中选择商品对应的分类。正确的分类可以帮助买家更快找到您的商品。

**第四步：填写商品描述**

在「商品描述」文本框中，详细介绍您的商品。可以包括：产地信息、种植/养殖方式、特点优势、食用建议等。好的描述能够增加商品的吸引力。

**第五步：设置价格**

在「价格」输入框中填写商品单价。价格应该合理定价，既要有竞争力，又要保证利润。

**第六步：设置库存**

在「库存数量」输入框中填写可售数量。库存不足时，商品会自动显示“仅剩X份”的标签。

**第七步：发布商品**

填写完所有信息后，点击「🚀 发布商品」按钮。商品即可上架销售！系统会显示“商品发布成功！”的提示信息。

### 3.3 管理订单

1. 点击底部导航栏的「订单」图标
2. 在订单页面，您可以查看所有买家下单的记录
3. 当有买家下单时，订单会显示为「待处理」状态
4. 您可以根据实际发货情况，在线下完成交易
5. 买家确认收货后，订单状态会自动变更为「已完成」

## 四、功能说明

### 4.1 购物车（蛇皮袋）

购物车是平台的核心功能之一，用于存放您想要购买的商品。主要操作包括：

- **添加商品**：在商品页面点击「装进蛇皮袋」按钮
- **查看购物车**：点击底部导航栏中间的购物车图标
- **修改数量**：点击数量旁的「+」或「-」按钮
- **删除商品**：点击商品右下角的垃圾桶图标
- **结算订单**：点击「结算」按钮创建订单

### 4.2 数据存储

平台采用浏览器本地存储（LocalStorage）保存数据。这意味着：

- 您的购物车内容、订单记录会保存在浏览器中
- 刷新页面后数据不会丢失
- 更换浏览器或清除缓存后数据会重置
- 每个浏览器独立存储数据

### 4.3 模拟支付

当前版本采用模拟支付功能。结算时无需真实付款，订单会直接创建成功。这是测试版本的简化设计，正式版本将接入真实支付渠道。

## 五、常见问题

### 5.1 技术问题

**Q：页面显示不正常怎么办？**

A：请尝试以下解决方案：

1. 刷新页面（F5键或点击浏览器刷新按钮）
2. 清除浏览器缓存后重新打开
3. 更换其他浏览器（推荐Chrome或Edge）
4. 检查网络连接是否正常

**Q：图片无法上传怎么办？**

A：请检查以下几点：

1. 图片格式是否为JPG、PNG等常见格式
2. 图片大小是否超过5MB
3. 浏览器是否有访问本地文件的权限
4. 尝试更换其他图片

**Q：数据丢失怎么办？**

A：平台数据存储在浏览器本地存储中。如果数据丢失，可能是因为：

1. 清除了浏览器缓存
2. 更换了浏览器
3. 隐私模式下浏览

如需避免数据丢失，建议不要清除浏览器缓存。

### 5.2 使用问题

**Q：如何修改已发布的商品信息？**

A：当前版本暂不支持修改已发布商品。建议删除原商品后重新发布。

**Q：如何删除购物车中的商品？**

A：打开购物车后，点击商品右下角的垃圾桶图标即可删除。

**Q：订单可以取消吗？**

A：当前版本暂不支持取消订单。如需取消，请联系平台管理员。

### 5.3 账号问题

**Q：需要注册账号才能使用吗？**

A：当前版本无需注册即可使用。所有数据存储在浏览器本地。

**Q：如何切换买家/卖家身份？**

A：平台不区分买家和卖家身份。所有用户既可以购买商品，也可以发布商品。点击「支棱起来」即可发布商品，点击购买按钮即可购买商品。
